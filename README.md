# 🎓 Tech Quiz Master

A dynamic, full-stack quiz web application built using the **MERN stack**. The platform allows users to test their technical knowledge across multiple web technologies with varying difficulty levels, featuring user authentication, real-time feedback, and performance tracking.

🚀 **Live Demo:** [https://quiz-app-six-nu-78.vercel.app/](https://quiz-app-six-nu-78.vercel.app/)

---

## 🌟 Features

- **Multi-Technology Coverage:** Interactive quizzes covering essential core technologies including HTML, CSS, JavaScript, React, and more.
- **Tiered Difficulty Levels:** Challenges categorized into Basic, Intermediate, and Advanced levels to suit different stages of learning.
- **User Authentication:** Secure user signup and login to save and track personalized progress.
- **Instant Performance Feedback:** Real-time scoring system and deep performance analytics available immediately upon quiz completion.
- **Responsive Dashboard:** Sleek, modern user interface optimized for seamless use across mobile, tablet, and desktop devices.

## 🚀 Getting Started

Follow these instructions to get a copy of the project up and running on your local machine for development and testing purposes.

### Prerequisites

Make sure you have Node.js and npm installed on your system.
- [Download Node.js](https://nodejs.org/)

### Installation

1. **Clone the repository:**
```bash
   git clone [https://github.com/YOUR_GITHUB_USERNAME/YOUR_REPOSITORY_NAME.git](https://github.com/YOUR_GITHUB_USERNAME/YOUR_REPOSITORY_NAME.git)
   cd YOUR_REPOSITORY_NAME
2.Set up the Backend:
Navigate to the server folder:
cd server # or backend, depending on your folder name
     ```
- Install dependencies:
```bash
     npm install
     ```
- Create a `.env` file in the root of your server directory and supply your environment variables:
```env
     PORT=5000
     MONGO_URI=your_mongodb_connection_string
     JWT_SECRET=your_jwt_secret_key
     ```
- Start the server:
```bash
     npm start
     ```

3. **Set up the Frontend:**
- Open a new terminal window and navigate to the frontend folder:
```bash
     cd client # or frontend
     ```
- Install dependencies:
```bash
     npm install
     ```
- Start the React development server:
```bash
     npm run dev # if using Vite, or 'npm start' if using Create React App
     ```

The app should now be running locally on `http://localhost:5173` (Vite) or `http://localhost:3000` (CRA)!
